# 🤖 MotivBot - Instagram Reels Avtomatik Bot

Har kuni Instagram'ga motivatsion videolar joylab beruvchi avtomatik bot.

## 🚀 Ishlash prinsipi

1. `facts.json` dan tasodifiy iqtibos tanlanadi
2. Chiroyli gradient fon + animatsiyali video yaratiladi
3. Edge-TTS orqali ovoz qo'shiladi
4. Video `output/video.mp4` va caption `output/caption.txt` ga saqlanadi
5. GitHub Actions orqali avtomatik Instagram'ga joylanadi
